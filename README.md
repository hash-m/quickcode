# quickcode
A simple and lightweight tool for generating QR codes and barcodes, built to reduce friction in my retail job.

Built with Flutter and a local database using Isar for persistant storage.

# Features
- Generate QR codes and barcodes from text input
- Save generated codes to local history
- Reuse previous entries from history
- Persistent storage using Isar

# Installation Guide

Prerequisites
- Flutter SDK (3.0 or above)
- Xcode (for macOS/iOS)
- CocoaPods


Guide
1. Clone the repository
2. Install dependencies
3. To run on MacOS: flutter run -d macos
4. To run on iOS: flutter run -d ios
5. To run on a browser: flutter run -d web-server
(I haven't tested this on Android yet)
